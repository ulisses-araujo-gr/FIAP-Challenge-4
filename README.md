# FIAP-Challenge-4
Trabalho de pós graduação FIAP

# Predição de Obesidade com Machine Learning

Este projeto foi desenvolvido com o objetivo de criar uma solução baseada em
Machine Learning para auxiliar a equipe médica na identificação de possíveis
níveis de obesidade, a partir de dados físicos e hábitos de vida dos indivíduos.

# Objetivo
Desenvolver um modelo preditivo com alta assertividade e disponibilizá-lo por
meio de uma aplicação interativa, além de construir uma visão analítica para
apoio à tomada de decisão.

# Base de Dados
A base de dados utilizada contém informações como idade, peso, altura, hábitos
alimentares, nível de atividade física e comportamento, tendo como variável alvo
o nível de obesidade.

# Pipeline de Machine Learning
O projeto seguiu uma pipeline completa de Machine Learning, incluindo:
- Análise exploratória dos dados
- Tratamento e codificação de variáveis categóricas
- Feature engineering
- Treinamento de modelo supervisionado
- Avaliação de desempenho

O modelo alcançou uma acurácia de aproximadamente **92,9%**, superando o
requisito mínimo de 75% estabelecido no desafio.

# Aplicação Preditiva
Foi desenvolvida uma aplicação utilizando **Streamlit**, permitindo que usuários
insiram seus dados e recebam a previsão do nível de obesidade de forma clara e
intuitiva. A aplicação atua como uma ferramenta de apoio à decisão e não
substitui o diagnóstico médico.

# Dashboard Analítico
Além da aplicação preditiva, foi construído um dashboard analítico no **Power BI**
para análise dos principais fatores associados à obesidade, fornecendo insights
relevantes para a equipe médica.

# Tecnologias Utilizadas
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Power BI
